# User Guide - HR Analytics Platform

## Getting Started

Welcome to the HR Analytics Platform! This guide will help you navigate and use all the features of the application.

### Logging In

1. Open the application in your web browser
2. Enter your username and password
3. Click "Sign In"
4. Your session will remain active for 30 minutes

**Note**: For security, you will be automatically logged out after 30 minutes of inactivity.

## Dashboard Overview

The main dashboard displays key HR metrics and analytics at a glance.

### Dashboard Components

#### KPI Cards
- **Total Headcount**: Current number of active employees
- **Active Employees**: Employees currently working
- **On Leave**: Employees on approved leave
- **New Hires**: Employees hired in the current period
- **Departures**: Employees who have left

Each card shows:
- Current value
- Trend indicator (up/down arrow)
- Percentage change from previous period

#### Performance Analytics
- Distribution of employee performance ratings
- Performance trends over time
- Performance vs. salary analysis
- Department performance comparison

#### Workforce Metrics
- Headcount by department
- Headcount by region
- Employment status breakdown
- 12-month historical trends

#### Turnover Analysis
- Historical turnover rates
- Turnover by department and region
- Turnover trends and patterns
- 6-month turnover predictions

#### Hiring Forecasts
- 12-month hiring predictions
- Predicted hires by department and role
- Confidence levels for predictions
- Critical roles with high turnover risk

## Employee Management

### Viewing Employees

1. Navigate to the **Employees** section
2. The employee table displays all employees with:
   - Name
   - Department
   - Position
   - Performance rating
   - Hire date
   - Status

### Searching Employees

**Basic Search**
1. Click the search box at the top of the employee table
2. Type the employee name or ID
3. Results update automatically as you type

**Advanced Search**
- Use field-specific syntax: `department:Sales` or `status:Active`
- Combine multiple criteria: `department:Sales status:Active`

### Filtering Employees

1. Click the **Filters** button
2. Select filter criteria:
   - Department
   - Region
   - Employment Status
   - Performance Score
   - Hire Date Range
3. Click **Apply Filters**
4. To save filters for later use, click **Save Filter Preset**

### Sorting

Click any column header to sort:
- First click: Sort ascending
- Second click: Sort descending
- Third click: Remove sort

### Viewing Employee Details

1. Click on an employee row to expand details
2. View:
   - Full contact information
   - Employment history
   - Performance history
   - Leave balance
   - Documents

## Analytics & Reports

1. Navigate to **Analytics** → **Performance**
2. View:
   - Performance distribution chart
   - Performance trends over time
   - Performance ratings breakdown
   - Performance vs. salary scatter plot
   - Department performance comparison

**Drill-Down**: Click on a department to see individual employee performance

### Workforce Analytics

1. Navigate to **Analytics** → **Workforce**
2. View:
   - Total headcount
   - Active employees
   - Employees on leave
   - New hires
   - Departures
3. Segment by:
   - Department
   - Region
   - Employment status

### Turnover Analysis

1. Navigate to **Analytics** → **Turnover**
2. View:
   - Historical turnover rates
   - Turnover by department
   - Turnover by region
   - Turnover trends
   - 6-month predictions with confidence levels

### Hiring Forecasts

1. Navigate to **Analytics** → **Hiring Forecast**
2. View:
   - 12-month hiring predictions

   - Predicted hires by role
   - Confidence levels
   - Critical roles with high turnover risk

**Adjust Parameters**:
1. Click **Adjust Parameters**
2. Modify:
   - Growth rate
   - Turnover assumptions

## Reports

### Generating Reports

1. Navigate to **Reports** → **Generate Report**
2. Select report type:
   - Performance Report
   - Workforce Report
   - Turnover Report
   - Hiring Forecast Report
3. Choose date range
4. Select filters (optional)
5. Click **Generate**

### Exporting Reports

After generating a report, choose export format:
- **PDF**: For printing and sharing
- **Excel**: For data analysis
- **CSV**: For importing to other systems

**Multi-Language Export**:
- Reports automatically export in your selected language
- Arabic reports include RTL formatting

### Saving Reports

1. Click **Save Report**
2. Enter report name
3. Choose save location
4. Click **Save**

Saved reports can be accessed from **Reports** → **Saved Reports**

## Preferences & Settings

### Language Settings

1. Click your profile icon (top right)
2. Select **Preferences**
3. Choose language:
   - English
   - العربية (Arabic)
4. Click **Save**

The interface will update immediately to your selected language.

### Dark Mode

1. Click the theme toggle (sun/moon icon) in the top navigation
2. Dark mode applies to all pages
3. Your preference is saved automatically

### Notification Preferences

1. Click your profile icon → **Preferences**
2. Select **Notifications**
3. Choose which notifications to receive:
   - Performance updates
   - Hiring alerts
   - System notifications
4. Click **Save**

### Filter Presets

1. Apply filters to the employee table
2. Click **Save Filter Preset**
3. Enter a name for the preset
4. Click **Save**

To use a saved preset:
1. Click **Load Preset**
2. Select the preset from the list
3. Filters apply automatically

## Troubleshooting

### I'm Logged Out Unexpectedly

- Sessions expire after 30 minutes of inactivity
- Log in again to continue
- Keep your browser tab active to maintain your session

### Data Not Updating

1. Check your internet connection
2. Look for the connection status indicator (top right)
3. If disconnected, the app will queue your changes and sync when reconnected
4. Refresh the page if needed

### Filters Not Working

1. Clear all filters by clicking **Clear Filters**
2. Reapply filters one at a time
3. Check that filter values are valid

### Export Not Working

1. Ensure you have sufficient disk space
2. Check browser download settings
3. Try a different export format
4. Clear browser cache and try again

### Performance Issues

1. Close other browser tabs
2. Clear browser cache
3. Disable browser extensions
4. Try a different browser
